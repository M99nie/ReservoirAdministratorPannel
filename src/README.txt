Данный проект был выполнен в рамках тестового задания студенческой IT Лаборатории в 2025 году.
В ходе проекта были использованы:
TypeScript (Язык программирования)
Zustand (для управления состояниями)
Axios* (HTTP-клиент)

В ходе работы была произведена вёрстка ВЕБ-страницы в соответствии с предоставленным макетом;
Настройка API для выполнения сетевых запросов;
(*в ходе работы с Axios получилось реализовать создание резервуара
и сохранение этих данных на локальном хосте, отображение списка
резервуаров по информации с сервера, однако внесение дальнейших изменений
по типу редактирования информации, или удаления резервуаров блокировались
со стороны сервера из-за CORS. По этой причине протестировать возможность редактирования
и удаления с моей стороны не предоставилось возможным)

Проект разбит на компоненты.
В папке API-файлы, отвечающие за взаимодействие с сервером.
reservoirs-содержит в себе операции из Axios и интерфейс резервуара.
reservoirStore-хранилище, реализованное с помощью Zustand.

В папке Components все компоненты странички
ConfirmDeleteWindow-окно подтверждения удаления резервуара
Header-заглавная часть странички
ReservoirList-список всех резервуаров, загружается с сервера
ReservoirScreen-отображает информацию о выбранном резервуаре,
так же служит для редактирования, сохранения, удаления и блокировки резервуаров
ReservoirTable-контейнер для компонентов ReservoirList и SearchWindow
ResScreenHeader-заголовочная часть ReservoirScreen
SearchWindow-компонент, предполагающийся для интеграции функции поиска резервуаров

В папке Pages все страницы проекта (одна)
HomePage-главная страница, на которой размещаются все компоненты

В папке Resources размещаются не программные ресурсы, необходимые для работы
Папка images содержит все изображения, используемые на страничке

В папке styles содержатся стили всех компонентов проекта.
Наименования стилей соответсвуют компонентам проекта.

Проект делался в IDE WebStorm 2024.3.4
Дата начала работы над проектом: 05.03.2025
